# Kolla Vishnu Charan - Portfolio

This is a personal portfolio website for Kolla Vishnu Charan, showcasing education, experience, skills, and projects.

## How to View

Open `index.html` in any modern web browser.

## Structure

- `index.html`: The main HTML file containing the portfolio content.
- `styles.css`: CSS file for styling the portfolio.

## Customization

To customize the portfolio:
- Edit `index.html` to update personal information, education, experience, etc.
- Modify `styles.css` to change colors, fonts, or layout.

## Deployment

You can deploy this static site to any web hosting service like GitHub Pages, Netlify, or Vercel by uploading the files.

## Projects

The portfolio includes:
- One academic project: Fake Review Detection in E-Commerce.
- All public GitHub repositories from the user's account, with links to each repo.
